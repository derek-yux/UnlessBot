### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```

3. Add your API Key!

   ```
   Get yours here: https://platform.openai.com/api-keys
   ```
